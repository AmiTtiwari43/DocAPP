4. DYNAMIC FEATURES & REAL-TIME UPDATES
================================================================================

A. Doctor Approval Workflow:
   - Doctor creates/edits profile -> "Request Verification" button
   - Status: Pending -> Admin sees in queue -> Approve/Reject
   - On approval: Doctor gets notification, profile appears in search
   - Real-time status updates in doctor dashboard

B. Review System:
   - Patient completes appointment -> Auto-prompt review modal
   - Review submission -> Instant update to doctor profile
   - Average rating recalculated automatically
   - Top doctors list updates dynamically
   - Doctor can reply to reviews (instant update)

C. Filtering & Search:
   - All filters work in real-time (no page reload)
   - URL params sync with filters
   - Debounced search input
   - Animated filter application

D. Notifications:
   - Toast notifications for all actions (Shadcn Toast)
   - Notification center (bell icon) with unread count
   - Real-time updates via polling or WebSocket (polling for now)

E. Booking Flow:
   - Date selection -> Fetch available slots -> Animate slot grid
   - Booking confirmation -> Toast notification
   - Appointment appears instantly in dashboard
